## 빌드 실패 시

### 일반적인 오류

1. **"Build Error"**
   - 환경 변수 누락 가능성
   - 빌드 로그 확인

2. **"Module not found"**
   - 패키지 설치 문제
   - `package.json` 확인

3. **"Environment Variable Missing"**
   - 환경 변수가 추가되지 않음
   - Vercel 설정에서 확인

### 해결 방법

1. **빌드 로그 확인**
   - Vercel 대시보드에서 상세 로그 확인
   - 오류 메시지 확인

2. **환경 변수 확인**
   - 모든 환경 변수가 추가되었는지 확인
   - Production, Preview, Development 모두에 추가되었는지 확인

3. **로컬 빌드 테스트**
   ```bash
   npm run build
   ```
   - 로컬에서 빌드가 성공하는지 확인
